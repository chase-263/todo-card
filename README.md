# Todo Item Card — HNG Stage 0

A clean, modern Todo / Task Card component built with vanilla HTML, CSS, and JavaScript.

## Live Demo
[View Live](https://chase-263.github.io/todo-card/)

## Features
- Displays a todo task with title, description, priority, status, due date and tags
- Checkbox toggles task completion — strikes through title and changes status to "Done"
- Time remaining calculates automatically from due date and updates every 60 seconds
- Fully responsive from 320px to 1200px
- Accessible — keyboard navigable, screen reader friendly, WCAG AA compliant

## Tech Stack
- HTML5 (semantic)
- CSS3 (no frameworks)
- Vanilla JavaScript (no frameworks)
